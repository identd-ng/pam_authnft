# Code of Conduct

Be respectful. Discuss code, not people. Assume good faith and ask
clarifying questions before escalating.

Unacceptable: harassment, personal attacks, sustained disruption of
discussion, or publishing others' private information without consent.

Enforcement is at the maintainer's discretion and may include edits,
locking threads, or bans. Report concerns privately via GitHub Security
Advisories on `identd-ng/pam_authnft` or by email to the maintainer listed
in the repository metadata.
